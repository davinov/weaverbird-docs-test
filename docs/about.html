WeaverBird is a visual query builder tool for transforming data.

Please take a look at the user interface documentation for more details.

If you're developer and wish to either embed it in one of your projects or just contribute, you can read the technical documentation.

Weaverbird was originally made by Toucan Toco. If you want to get more information about our main product check it here
